Spell Checker

To invoke the Spell Checker, click the buttons as illlustrated in the image below.

This demo shows how the DevExpress ASP.NET Spell Checker (ASPxSpellChecker) control’s spell check functinality can be used in the ASPxHtmlEditor's Desin View to check the spelling of the editor content.

The ASPxSpellChecker control proviedes a straigtforward way to add Microsoft Office-style spell checking capabillities to your next ASP.NET aplication. With this control, you only need to write a single line of code to check text within an individual text editor, or within all controls in a specifid container. When an error is encountered, you can review it and take the necessary actions, using a Microsoft Outlook-style dialog.
